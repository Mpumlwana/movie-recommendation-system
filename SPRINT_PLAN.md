# Sprint Plan For a Movie Recommendation System

---

## Sprint Duration
2 Weeks

---

## Sprint Goal
To Deliver a Minimum Viable Product (MVP) that allows users to log in, search for movies, and receive personalized recommendations.

---

## Selected User Stories

- US-001 Login/Register  
- US-002 Search Movies  
- US-003 Rate Movies  
- US-004 Get Recommendations  
- US-005 View Details

## Contribution to MVP

This sprint delivers the core features that are required for the system to function: authentication, movie search, rating, and recommendations. These features directly address primary user needs and form the foundation of the system.
